# test-coverage-mcp

🔬 A provider-extensible MCP server for test coverage intelligence.

This is the core package of the test-coverage-mcp monorepo.

## Features

- Provider-extensible architecture
- Capability-driven tool contract
- MCP server with FastMCP
- Web server with FastAPI
- Multiple transport support (stdio, HTTP)

## Installation

```bash
pip install test-coverage-mcp
```

## Quick Start

```bash
# Start MCP server
test-coverage-mcp serve

# Start web server
test-coverage-mcp web
```
