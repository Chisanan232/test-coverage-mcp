# Getting Started with test-coverage-mcp

test-coverage-mcp is an intelligent MCP server for test coverage analysis. This guide will help you install, configure, and start using test-coverage-mcp for coverage analysis and recommendations.

## What's Included

✅ **MCP Server**: Model Context Protocol server with 11 coverage analysis tools
✅ **Provider Support**: Extensible architecture supporting Codecov, Coveralls, and custom providers
✅ **Multiple Transports**: stdio, HTTP, and SSE transport options
✅ **Web API**: FastAPI-based REST interface for HTTP integration
✅ **Docker Support**: Production-ready containerization
✅ **Comprehensive Documentation**: Complete guides, API reference, and examples
