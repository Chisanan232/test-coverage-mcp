# Running test-coverage-mcp

This guide explains how to run, configure, and use test-coverage-mcp.

## Step 1: Configure Coverage Providers

Before running test-coverage-mcp, configure at least one coverage provider:

### Codecov Provider Setup

```bash
# Set environment variables
export CODECOV_API_TOKEN="your-codecov-token"
export CODECOV_OWNER="your-org"
export CODECOV_REPO="your-repo"

# Or create .env file
cat > .env << EOF
CODECOV_API_TOKEN=your-codecov-token
CODECOV_OWNER=your-org
CODECOV_REPO=your-repo
EOF
```

See [Configuration Guide](../guides/configuration.md) for detailed setup.

## Step 2: Start the MCP Server

### Using stdio Transport (Recommended for Claude)

```bash
# Start server with stdio transport
test-coverage-mcp serve --transport stdio

# Or with uv
uv run test-coverage-mcp serve --transport stdio
```

### Using HTTP Transport

```bash
# Start server with HTTP transport
test-coverage-mcp serve --transport http --host 0.0.0.0 --port 8000

# Or with uv
uv run test-coverage-mcp serve --transport http --host 0.0.0.0 --port 8000
```

### Using SSE Transport

```bash
# Start server with SSE transport
test-coverage-mcp serve --transport sse --host 0.0.0.0 --port 8000
```

## Step 3: Using MCP Tools

Once the server is running, you can use the MCP tools:

### List Available Providers

```bash
# Using MCP client
tool: coverage.providers.list
parameters:
  include_capabilities: true
  include_health: true
```

### Get Repository Health

```bash
tool: coverage.repository.health
parameters:
  repo_slug: "owner/repo"
  threshold: 80.0
```

### Compare Coverage Between Branches

```bash
tool: coverage.comparison.refs
parameters:
  repo_slug: "owner/repo"
  base_ref: "main"
  head_ref: "feature-branch"
```

See [MCP Tools Overview](../mcp-tools/overview.md) for complete tool documentation.

## Step 4: Development Workflow

### Running Tests

```bash
# Run all tests
uv run pytest

# Run tests with coverage
uv run pytest --cov=test-coverage-mcp/src --cov-report=html

# Run specific test file
uv run pytest test-coverage-mcp/test/unit_test/test_tools.py
```

### Code Quality Checks

```bash
# Format code with black
uv run black test-coverage-mcp/src test-coverage-mcp/test

# Lint code with ruff
uv run ruff check test-coverage-mcp/

# Type checking with mypy
uv run mypy test-coverage-mcp/src

# Run all pre-commit hooks
uv run pre-commit run --all-files
```

### Adding Dependencies

```bash
# Add production dependencies
uv add requests

# Add development dependencies
uv add --dev pytest-mock

# Add to specific dependency group
uv add --group test pytest-asyncio
```

## Step 5: Docker Deployment

### Build Docker Image

```bash
# Build image
docker build -t test-coverage-mcp:latest .

# Or with specific Python version
docker build --build-arg PYTHON_VERSION=3.13 -t test-coverage-mcp:latest .
```

### Run Docker Container

```bash
# Run with stdio transport
docker run -e CODECOV_API_TOKEN=your-token test-coverage-mcp:latest

# Run with HTTP transport
docker run -p 8000:8000 \
  -e CODECOV_API_TOKEN=your-token \
  test-coverage-mcp:latest \
  serve --transport http --host 0.0.0.0 --port 8000
```

### Using Docker Compose

```bash
# Start services
docker-compose up

# Stop services
docker-compose down
```

## Step 6: Integration Examples

### Python Integration

```python
from test_coverage_mcp.mcp_server.tools import (
    list_coverage_providers,
    get_repository_test_health,
    compare_coverage_between_refs
)

# List providers
providers = list_coverage_providers()

# Check repository health
health = get_repository_test_health(
    repo_slug="owner/repo",
    threshold=80.0
)

# Compare coverage
comparison = compare_coverage_between_refs(
    repo_slug="owner/repo",
    base_ref="main",
    head_ref="feature-branch"
)
```

### CI/CD Integration

```yaml
# GitHub Actions example
- name: Check Coverage
  run: |
    test-coverage-mcp serve --transport stdio << 'EOF'
    {
      "jsonrpc": "2.0",
      "id": 1,
      "method": "tools/call",
      "params": {
        "name": "coverage.repository.health",
        "arguments": {
          "repo_slug": "owner/repo"
        }
      }
    }
    EOF
```

## Step 7: Troubleshooting

### Provider Not Found

```bash
# List available providers
test-coverage-mcp list-providers

# Check provider health
test-coverage-mcp health-check --provider codecov
```

### Configuration Issues

```bash
# Verify configuration
test-coverage-mcp config validate

# Show current configuration
test-coverage-mcp config show
```

### Connection Issues

```bash
# Test provider connectivity
test-coverage-mcp test-provider --provider codecov

# Check API token
test-coverage-mcp verify-token --provider codecov
```
