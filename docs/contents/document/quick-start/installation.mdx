# Installing test-coverage-mcp

This guide provides step-by-step instructions for installing and setting up test-coverage-mcp.

## Method 1: Install from PyPI (Recommended)

The easiest way to install test-coverage-mcp is from PyPI:

```bash
# Install core package
pip install test-coverage-mcp

# Or with uv
uv pip install test-coverage-mcp
```

## Method 2: Development Installation

For development or contributing to test-coverage-mcp:

### Step 1: Clone the Repository

```bash
git clone https://github.com/Chisanan232/test-coverage-mcp.git
cd test-coverage-mcp
```

### Step 2: Install Development Dependencies

<Tabs>
<TabItem value="uv" label="Using uv (Recommended)">

```bash
# Install all dependencies including dev tools
uv sync

# Or install only production dependencies
uv sync --no-dev
```

</TabItem>
<TabItem value="pip" label="Using pip">

```bash
# Create virtual environment
python -m venv .venv
source .venv/bin/activate  # On Windows: .venv\Scripts\activate

# Install in development mode
pip install -e ".[dev]"
```

</TabItem>
</Tabs>

## Installing Coverage Providers

test-coverage-mcp supports multiple coverage providers. Install the providers you need:

```bash
# Install Codecov provider
pip install test-coverage-mcp-codecov

# Install Coveralls provider (when available)
pip install test-coverage-mcp-coveralls

# Or install all providers
pip install test-coverage-mcp[all-providers]
```

## Configuration

After installation, configure your coverage providers:

```bash
# Set environment variables for your provider
export CODECOV_API_TOKEN="your-token-here"
export CODECOV_OWNER="your-org"
export CODECOV_REPO="your-repo"
```

See the [Configuration Guide](../guides/configuration.md) for detailed setup instructions.

## Verification

Verify your installation by running:

```bash
# Check test-coverage-mcp installation
python -c "import test_coverage_mcp; print(test_coverage_mcp.__version__)"

# Check Python environment
python --version

# Run tests to ensure everything works (development installation only)
uv run pytest
```
